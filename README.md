# 📚 考研英语单词游戏

一个帮助考研学子轻松背单词的互动游戏网站，支持发音、多种游戏模式、进度追踪，可以分享给朋友一起学习！

## ✨ 功能特色

- 🎯 **四种游戏模式**：选择题、闪卡记忆、拼写挑战、听力训练
- 🔊 **真人发音**：使用 Web Speech API，每个单词都有标准美式发音
- 📊 **进度追踪**：自动记录学习数据，正确率、连击数一目了然
- 🔥 **连击系统**：连续答对获得成就感和动力
- 📤 **分享功能**：一键分享给朋友，一起背单词
- ⚙️ **可定制**：选择难度等级、调整每轮题数

## 🎮 游戏模式

| 模式 | 说明 |
|------|------|
| 🎯 选择题 | 看英文，从四个选项中选出正确的中文含义 |
| 🃏 闪卡 | 翻转卡片，自测是否记住了单词 |
| ✍️ 拼写 | 看中文，拼写对应的英文单词 |
| 🎧 听力 | 听发音，选择对应的英文单词 |

## 🚀 快速开始

## 🛠 本地开发

```bash
# 克隆仓库
git clone https://github.com/agwen789-maker/minigame.git
cd minigame

# 直接打开 index.html 即可运行
# 或者使用本地服务器
npx serve .
```

## 📦 技术栈

- 纯 HTML + CSS + JavaScript（零依赖）
- Web Speech API（发音）
- localStorage（数据持久化）
- GitHub Pages（部署）

## 📝 词库

内置 **856 个**考研英语高频词汇，按难度分为三级：
- 🌱 基础（Level 1）：最常用词
- 🌿 进阶（Level 2）：核心词汇
- 🌳 挑战（Level 3）：高阶词汇
